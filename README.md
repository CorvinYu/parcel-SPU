# 📦 取件码海大版 | Pickup Code Haida Edition

---

这是一个基于原开源项目二次维护的海大版分支，当前主要面向上海海事大学快递站场景做优化。

当前版本保持免费、开源、无广告、不联网的设计方向，不收集个人信息。

本 app 会自动解析收到的短信，并从中提取地址和取件码信息，展示到桌面卡片上。

当前已完成的海大版调整：

- 页面背景支持自定义图片
- 应用名称、版本信息、关于页说明已与原项目区分

后续如果发现更适合上海海事大学快递站的优化方向，会继续在这个版本中补充。

---

# ✨ 功能

- 自动解析短信中的取件码与地址
- 支持桌面小组件展示
- 支持自定义规则改进解析效果
- 支持淘宝身份码和拼多多身份码
- 支持监听第三方 app 通知并自动保存取件信息
- 支持地址归类
- 支持老人模式
- 支持页面背景自定义图片

# ❓ 使用说明

1. 桌面卡片添加
   一般在系统全部卡片、插件或安卓小组件列表中。

2. 页面背景
   可在 app 内进入“页面背景”，选择自定义图片并调整遮罩强度、模糊和显示范围。

3. 小米手机权限
   需要打开通知类短信权限：`权限管理 -> 其他权限 -> 通知类短信 -> 始终允许`

4. 无法识别的短信
   某些没有发送者号码的服务信息短信可能无法识别。可以关闭相关服务信息短信，或开启第三方通知监听作为补充。

5. 桌面卡片不更新
   可以尝试打开通知监听权限，并在系统里设置自启动、后台不受限，再重新添加桌面卡片。
